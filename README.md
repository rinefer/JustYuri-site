Интерактивный веб-проект, созданный по мотивам визуальной новеллы Doki Doki Literature Club (DDLC). Проект представляет собой последовательность мини-игр и текстовых уровней, объединенных общим сюжетом и атмосферой.

### Описание проекта
Данный сайт был разработан как творческий эксперимент и дань уважения оригинальной игре. Основной акцент сделан на передачу специфического стиля повествования и визуальных эффектов через веб-технологии. Проект включает в себя систему авторизации, уровни с составлением фраз и финальную интерактивную сцену.

### Структура репозитория
* `index.html` — главный файл приложения, содержащий разметку всех игровых экранов (экран входа, сад, игра в слова).
* `style.css` — файл стилей, отвечающий за визуальное оформление, шрифты и эффекты переходов.
* `script.js` — программный код, управляющий логикой переключения уровней, проверкой паролей и обработкой прогресса игрока.
* `assets/` — папка с аудиофайлами и графическими ресурсами (если они используются).

### Установка и запуск
Для локального запуска достаточно клонировать репозиторий и открыть файл index.html в любом современном браузере. Дополнительная сборка или установка зависимостей не требуется.
